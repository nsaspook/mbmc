Chart Demo (using SVG painter)

Demo of the chart component using the SVGChartPainter painter implementation.

Tested in Mozilla Firefox and Opera 9 however any browser with JS, DOM and SVG or SVG Tiny support should be able to handle it. Mozilla requires content to be served as text/xml or application/xhtml+xml to allow inline SVG.

This implementation is not complete and is provided mostly as a reference at this stage.
